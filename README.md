**SIEL LEAGUE APP**

This project is a React web application that displays information about basketball games. The application allows the user to filter games based on search input and based on the home and away team. It also allows the user to add new games to the list.

**Prerequisites**


Before you can run the application, you need to have Node.js installed on your computer.

**Installation**


Clone the repository to your local machine.
Navigate to the project directory using the command prompt.

Run npm install to install the dependencies.

Run npm start to start the application.
Usage

The application displays a table of basketball games. You can filter the games by typing in the search input or by selecting the home or away team. You can also add new games to the list by clicking the "Add Game" button and filling out the form.

To start the backend server, the user needs to run the command json-server --watch db.json --port 4000. This command will start the server and make the data available through the API endpoints.

**Files**

App.js: The main component of the application.
Header.js: The component that displays the header.
GamesTable.js: The component that displays the table of games.
GamesForm.js: The component that displays the form for adding new games.
App.css, Header.css, GamesTable.css, and GamesForm.css: The CSS files for styling the components.
Dependencies
react: The JavaScript library used for building the user interface.
react-dom: The package that provides DOM-specific methods for working with React.
react-scripts: The package that includes scripts and configuration used by Create React App.
prop-types: The package that provides runtime type checking for React props.
bootstrap: The package that provides pre-built CSS and JavaScript components for building responsive, mobile-first web sites and applications.
axios: The package that provides a simple HTTP client for making requests to the API.
